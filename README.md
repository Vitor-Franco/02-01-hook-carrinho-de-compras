## 🚀 Getting started

Clone o projeto e acesse a pasta

```bash
$ git clone https://github.com/Vitor-Franco/02-01-hook-carrinho-de-compras.git && cd 02-01-hook-carrinho-de-compras
```

Siga o passo-passo abaixo.

```bash
# Instale as dependências
$ yarn

# Rode a fake API
$ yarn server

# Start o projeto
$ yarn start
```
